# Posicionamiento
En esta tarea diseñaremos una página web sencilla utilizando cajas (div) para demostrar el uso correcto de las técnicas de posicionamiento en CSS vistas en clase.

Deberá de estar compuesta por varios bloques o cajas, utilizando HTML y CSS, donde se apliquen diferentes tipos de posicionamiento estudiados en el tema.

# Tenéis libertad total en el diseño, pero la página debe demostrar claramente que sabéis utilizar y diferenciar los siguientes conceptos:
position: static
position: relative
position: absolute
position: fixed
Uso de top, left, right y bottom
float y clear

# La página debe cumplir como mínimo con lo siguiente:
Un contenedor principal que organice el contenido de la página. Al menos:
Un elemento con posicionamiento relativo
Un elemento con posicionamiento absoluto respecto a otro elemento
Un elemento con posicionamiento fijo visible aunque se haga scroll

Uso de float para colocar uno o más elementos a la izquierda o derecha.

Uso de clear para evitar problemas provocados por elementos flotantes.

Aplicación correcta de propiedades top, left, right o bottom en los elementos que lo requieran.
